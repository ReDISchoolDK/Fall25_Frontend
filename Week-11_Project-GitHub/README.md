# Week 11: Final Project Showcase & Deployment

## Summary

In the final week, you'll put the finishing touches on your project and deploy it online. Hosting on GitHub Pages will let you share your work with the world.

## Tasks

### Project Work

- **Project Finalization:**
  - Ensure all features are implemented and functioning as intended.
  - Review your project: ensure the code is tidy, comments are added where necessary, and the project is well-documented with a `README.md`.
  - Make sure your project is in a complete and presentable state.

- **Deployment:**

  - Deploy your project to GitHub Pages (via repository settings).
  - Verify that all assets load correctly and the project runs as expected on the live site.
  - Add the live link to your project's `README.md`.

- **Showcase Preparation:**
  - Prepare a brief presentation or demo of your project.
  - Highlight key features, challenges you overcame, and what you learned during development.
